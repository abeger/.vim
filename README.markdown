Installation
------------

### Install Modules:

    cd ~
    git clone git://github.com/abeger/.vim.git
    ln -s .vim/vimrc .vimrc
    cd .vim
    git submodule init
    git submodule update   

### Install Ack (if necessary):
See [ack.vim](https://github.com/mileszs/ack.vim) instructions.
